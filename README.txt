- Run the 'run.sh <hdfs_input_path>' to run the program with parameterized hdfs path
- Run the 'run_devset.sh' to run the program with hardcoded hdfs path to devset file
- Run the 'run_full.sh' to run the program with hardcoded hdfs path to full dataset

After the processing is done, the output content will be in a newly created file 'output.txt'
